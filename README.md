# ⚡ Project Omni: Local Agentic CLI

Omni adalah *AI-powered CLI tool* lokal yang menjembatani repositori kode dengan Google Gemini 2.5 Flash API. Dirancang dengan prinsip *Zero-Cost* dan *Clean Architecture*, alat ini mengeliminasi kebutuhan *copy-paste* manual ke antarmuka web, memungkinkan alur kerja *vibe coding* langsung dari terminal.

## 🎯 Core Features
* **Context-Aware Code Generation:** Membaca struktur direktori dan file lokal sebagai konteks untuk LLM.
* **Terminal Native:** Interaksi mulus menggunakan `Typer` dan `Rich` untuk UI terminal yang elegan.
* **Zero-Cost:** Memanfaatkan Free Tier API tanpa mengorbankan kualitas dan privasi basis kode.

## 🛠️ Tech Stack
* **Python 3.11+**
* **Google Generative AI (Gemini 2.5 Flash)**
* **Typer** (CLI Framework)
* **Rich** (Terminal formatting)
